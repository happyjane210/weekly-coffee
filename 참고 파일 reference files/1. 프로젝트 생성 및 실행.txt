// 프로젝트 생성
npx create-next-app@latest 프로젝트명 --ts
프로젝트명 입력

// 프로젝트 실행(개발모드)
npm run dev

// 프로젝트 빌드
npm run build

// 프로젝트 실행(프로덕션모드)
npm start




===========================

// create-next-app 으로 설치하면
  1. 컴파일과 번들링이 자동으로됨 (webpack과 babel)
  2. 자동 리스페쉬 기능으로 수정하면 화면에 바로 반영됨
  3. 서버사이드 렌더링이 지원됨
  4. 스테틱 파일을 지원함.


// public 폴더는 정적파일을 저장할 수 있음 
  정적파일 = 스테틱파일
