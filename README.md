# weekly-coffee 주간커피

여러 로스터리 사업장과 소비자를 연결하는, <br>
원두 정기 구독 서비스를 제공합니다.

<br>

## 담당 개발 서비스 - 구독 서비스 (client)

다양한 원두를 소비자가 원하는 옵션에 맞춰 정기 구독 할 수 있는 구독 서비스를 개발 합니다

- 판매 제품 등록
- 주문 관리
- 마이페이지, 공지사항, 장바구니
